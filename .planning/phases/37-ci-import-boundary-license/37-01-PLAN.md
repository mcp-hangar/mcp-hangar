<task type="auto">
  <name>Task 1: Add enterprise-boundary job to pr-validation.yml</name>
  <files>.github/workflows/pr-validation.yml</files>
  <action>
Add a new `enterprise-boundary` job to `.github/workflows/pr-validation.yml` that runs `scripts/check_enterprise_boundary.sh`. This job must:

1. Run on `ubuntu-latest`
2. Check out the repo with `actions/checkout@v4`
3. Run `bash scripts/check_enterprise_boundary.sh`
4. NOT depend on `detect-changes` (it should always run, regardless of which files changed)

Add `enterprise-boundary` to the `required-check` job's `needs` array so it becomes a required passing check before merge.

The resulting pr-validation.yml structure should be:

```yaml
jobs:
  detect-changes:
    # ... (existing, unchanged)

  enterprise-boundary:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Enterprise import boundary check
        run: bash scripts/check_enterprise_boundary.sh

  summary:
    # ... (existing, unchanged)

  required-check:
    runs-on: ubuntu-latest
    needs: [detect-changes, enterprise-boundary]
    steps:
      - run: echo "PR validation complete"
```

Do NOT modify the existing detect-changes, summary, or other jobs beyond adding enterprise-boundary to required-check's needs.
Do NOT duplicate the script content inline -- reference the existing script.
  </action>
  <verify>
    <automated>python -c "
import yaml, sys
with open('.github/workflows/pr-validation.yml') as f:
    wf = yaml.safe_load(f)
jobs = wf.get('jobs', {})
# Check enterprise-boundary job exists
if 'enterprise-boundary' not in jobs:
    print('FAIL: enterprise-boundary job not found')
    sys.exit(1)
eb = jobs['enterprise-boundary']
# Check it has checkout + script step
steps = eb.get('steps', [])
step_runs = [s.get('run', '') for s in steps]
if not any('check_enterprise_boundary' in r for r in step_runs):
    print('FAIL: boundary check script not referenced in steps')
    sys.exit(1)
# Check required-check depends on enterprise-boundary
rc = jobs.get('required-check', {})
needs = rc.get('needs', [])
if 'enterprise-boundary' not in needs:
    print('FAIL: required-check does not depend on enterprise-boundary')
    sys.exit(1)
print('PASS: pr-validation.yml has enterprise-boundary job in required path')
