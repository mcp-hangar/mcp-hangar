    **Class: K8sNetworkMonitor**

    ```python
    class K8sNetworkMonitor:
        """Monitors network connections from K8s provider pods.

        Two strategies:
        1. Primary: Watch K8s Events for NetworkPolicy audit events
        2. Fallback: Exec into provider pods and read /proc/net/tcp

        Args:
            namespace: K8s namespace where provider pods run (default: "default").
            use_in_cluster: Whether to use in-cluster config (default: True).
        """

        def __init__(self, namespace: str = "default", use_in_cluster: bool = True) -> None:
            if not K8S_AVAILABLE:
                raise ImportError("kubernetes package required for K8s network monitoring")
            if use_in_cluster:
                try:
                    k8s_config.load_incluster_config()
                except Exception:
                    k8s_config.load_kube_config()
            else:
                k8s_config.load_kube_config()
            self._core_v1 = k8s_client.CoreV1Api()
            self._namespace = namespace
            self._resource_version: str | None = None
            self._audit_available: bool | None = None  # None = not yet checked
    ```

    **Method: poll_connections(provider_id: str, namespace: str | None = None) -> list[NetworkObservation]**

    1. Try `_poll_audit_events(provider_id, namespace)` if audit not known unavailable
    2. If audit returns empty or unavailable, fall back to `_poll_pod_exec(provider_id, namespace)`
    3. Convert results to NetworkObservation records

    **Method: _poll_audit_events(provider_id, namespace) -> list[tuple[str, int, str]]**

    1. List events in namespace with field selector `reason=NetworkPolicy*` or iterate recent events
    2. Filter events where `involved_object.name` matches a pod labeled `mcp-hangar.provider-id={provider_id}`
    3. Parse event message for destination IP:port (format varies by CNI)
    4. On first call, if no NetworkPolicy events found after checking, set `self._audit_available = False`
    5. Return list of (host, port, protocol)

    **Method: _poll_pod_exec(provider_id, namespace) -> list[tuple[str, int, str]]**

    1. List pods with label `mcp-hangar.provider-id={provider_id}` and status Running
    2. For each pod, exec `cat /proc/net/tcp` using `stream()` from kubernetes
    3. Parse output using `parse_proc_net_tcp` from proc_net_parser (lazy import)
    4. Per-pod fault barrier (try/except, log warning, continue)
    5. Return aggregated connections from all pods

    **Method: _find_provider_pods(provider_id, namespace) -> list**

    1. `self._core_v1.list_namespaced_pod(namespace, label_selector=f"mcp-hangar.provider-id={provider_id}")`
    2. Filter for Running phase
    3. Return pod objects
