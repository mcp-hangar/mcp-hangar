Use Recharts `BarChart` (horizontal, responsive) to display provider counts by state.
State → colour mapping:

- `ready`: `#22c55e` (green-500)
- `cold`: `#9ca3af` (gray-400)
- `initializing`: `#3b82f6` (blue-500)
- `degraded`: `#f59e0b` (amber-500)
- `dead`: `#ef4444` (red-500)

Transform `data: Record<string, number>` into `[{ state, count, fill }]` array for Recharts.
Use `<ResponsiveContainer width="100%" height={160}>` wrapping `<BarChart layout="vertical">`.
Axes: `<YAxis dataKey="state" type="category" width={80} />` and `<XAxis type="number" allowDecimals={false} />`.
Add `<Tooltip />` and a single `<Bar dataKey="count">` with `<Cell fill={entry.fill} />` for each bar.
If `data` is empty or all zeros, render `<EmptyState message="No provider data." className="py-6" />`.

Recharts imports:


**LiveEventFeed.tsx**
Props: `{ maxEvents?: number; className?: string }` (default maxEvents=20)
Uses `useEventStream({ bufferSize: maxEvents })` internally.
Renders:

- A header row: `"Live Events"` (text-sm font-medium) + WS status dot (green=connected, yellow=connecting, red=error/disconnected)

- If `events.length === 0`: show `<EmptyState message="Waiting for events..." className="py-4" />`
- Status dot colours: `connected` → `bg-green-400`, `connecting` → `bg-yellow-400 animate-pulse`, else → `bg-gray-300`
