**GREEN phase -- implement `packages/operator/pkg/networkpolicy/builder.go`:**

Package `networkpolicy`. Imports:
- `networkingv1 "k8s.io/api/networking/v1"`

- `mcpv1alpha1 "github.com/mcp-hangar/mcp-hangar/operator/api/v1alpha1"`

Functions to implement:

1. `BuildNetworkPolicy(provider *mcpv1alpha1.MCPProvider) *networkingv1.NetworkPolicy` -- main entry point.
   - Return nil if `provider.Spec.Capabilities == nil || provider.Spec.Capabilities.Network == nil`
   - Build ObjectMeta with `NetworkPolicyName(provider.Name)`, `provider.Namespace`, labels from `buildLabels(provider)`
   - Build Spec with PodSelector matching `mcp-hangar.io/provider: provider.Name`, PolicyTypes `[Egress]`, Egress from `buildEgressRules()`
   - Add annotation `mcp-hangar.io/host-warnings` listing host-only rules that lack CIDR enforcement

2. `NetworkPolicyName(providerName string) string` -- returns `"mcp-provider-" + providerName + "-egress"`

3. `buildLabels(provider) map[string]string` -- returns:
   - `app.kubernetes.io/managed-by: mcp-hangar-operator`
   - `mcp-hangar.io/provider: provider.Name`
   - `mcp-hangar.io/component: network-policy`

4. `buildEgressRules(caps *mcpv1alpha1.NetworkCapabilitiesSpec) []networkingv1.NetworkPolicyEgressRule`:
   - If `caps.DNSAllowed == nil || *caps.DNSAllowed == true` (default true): prepend DNS egress rule (UDP+TCP port 53)
   - If `caps.LoopbackAllowed != nil && *caps.LoopbackAllowed == true`: add `127.0.0.0/8` egress rule
   - For each EgressRuleSpec in caps.Egress: call `translateEgressRule()`

5. `translateEgressRule(rule mcpv1alpha1.EgressRuleSpec) networkingv1.NetworkPolicyEgressRule`:
   - If `rule.CIDR != ""`: add IPBlock peer with that CIDR
   - If `rule.CIDR == ""` and `rule.Host != ""`: NO IPBlock peer (host-only = port-level enforcement only)
   - If `rule.Port > 0`: add Ports with that port number, mapped protocol
   - If `rule.Port == 0`: omit Ports entirely (all ports allowed for that peer)

6. `mapProtocol(protocol string) *corev1.Protocol`:
   - "https", "http", "grpc", "tcp" -> `&corev1.ProtocolTCP`
   - "any" or "" -> `nil` (allows any protocol)

7. Helper: `dnsEgressRule() networkingv1.NetworkPolicyEgressRule` -- UDP+TCP port 53

8. Helpers: `hostWarnings(caps) string` -- comma-separated list of host-only rules for annotation

Run tests -- they MUST pass (GREEN). Commit implementation.
