**Reconcile loop (same flow as MCPProviderReconciler):**

1. Fetch MCPProviderGroup; return nil if NotFound
2. Check deletion timestamp; if set, call reconcileDelete (remove finalizer, clear metrics via ClearGroupMetrics)
3. Add finalizer if not present; update and requeue
4. Call reconcileNormal
5. Track metrics: ReconcileTotal (controller="mcpprovidergroup"), ReconcileDuration

**reconcileNormal method:**

1. Update ObservedGeneration if changed
2. Convert `group.Spec.Selector` to labels.Selector via `metav1.LabelSelectorAsSelector()`
   - If selector is nil, set Ready=Unknown reason "NoSelector" and return (ctrl.Result{}, nil)
   - If selector parsing fails, set Ready=False reason "InvalidSelector" with error message, return (ctrl.Result{}, nil)
3. List MCPProviders in same namespace matching selector via `client.MatchingLabelsSelector{Selector: selector}`
4. Aggregate status counts by iterating `providerList.Items`:
   - Switch on `provider.Status.State`: Ready -> ReadyCount, Degraded -> DegradedCount, Cold -> ColdCount, Dead -> DeadCount, default (Initializing or empty) -> ColdCount
   - Build `ProviderMemberStatus` slice with Name, Namespace, State, LastHealthCheck from each provider
   - Set ProviderCount = len(providerList.Items)
5. Set `ActiveStrategy = string(group.Spec.Strategy)`
6. Call evaluateConditions (see below)
7. Update metrics: GroupProviderCount gauge for each state
8. Status().Update the group
9. Return (ctrl.Result{RequeueAfter: readyRequeueAfter}, nil) on success

**evaluateConditions method (per locked decisions):**

Zero-member groups (ProviderCount == 0):

- Ready = Unknown, reason "NoProviders", message "No providers match selector"
- Available = False, reason "NoProviders"
- Degraded = False, reason "NoProviders"

Non-zero groups:

- Available = True if ReadyCount > 0 (at least 1 provider can serve), else False
- Ready = True if `status.IsHealthy(group.Spec.HealthPolicy)` returns true, else False
  - Use existing IsHealthy() helper which handles MinHealthyCount override vs MinHealthyPercentage
- Degraded = True if (DegradedCount + DeadCount) > 0 (coexists with Ready per locked decision), else False

**SetupWithManager method:**

```go
func (r *MCPProviderGroupReconciler) SetupWithManager(mgr ctrl.Manager) error {
    return ctrl.NewControllerManagedBy(mgr).
        For(&mcpv1alpha1.MCPProviderGroup{}).
        Watches(
            &mcpv1alpha1.MCPProvider{},
            handler.EnqueueRequestsFromMapFunc(r.findGroupsForProvider),
        ).
        Complete(r)
}
```

**findGroupsForProvider method:**
List all MCPProviderGroups in the provider's namespace. For each group, parse its selector and check if provider's labels match. If match, enqueue group for reconciliation.

**Imports needed (in addition to MCPProviderReconciler imports):**

- `"k8s.io/apimachinery/pkg/labels"` for labels.Set
- `"sigs.k8s.io/controller-runtime/pkg/handler"` for EnqueueRequestsFromMapFunc

**Do NOT:**

- Create owner references from group to providers (group is read-only aggregator per locked decision)
- Use Ginkgo/Gomega (testify is project convention)
- Import provider builder (group does not manage pods)
