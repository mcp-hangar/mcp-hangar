**Reconcile loop (same flow as MCPProviderReconciler):**

1. Fetch MCPDiscoverySource; return nil if NotFound
2. Check deletion; if set, call reconcileDelete (clean up owned MCPProviders via label selector, remove finalizer, clear metrics)
3. Add finalizer if not present
4. Call reconcileNormal
5. Track metrics: ReconcileTotal (controller="mcpdiscoverysource"), ReconcileDuration

**reconcileNormal method:**

1. **Paused check FIRST** (per locked decision and pitfall #7): If `source.IsPaused()`, set Paused condition to True, set Synced condition to Unknown reason "Paused", return (ctrl.Result{RequeueAfter: 5 * time.Minute}, nil). Do NOT scan, create, or delete anything.
2. Clear Paused condition (set to False) if not paused
3. Update ObservedGeneration
4. Parse RefreshInterval from spec (default 1m) using `time.ParseDuration`
5. Start sync timer
6. Call `discoverProviders(ctx, source)` which returns `(map[string]DiscoveredProviderInfo, []string, error)` where the string slice is a list of scan errors (partial failures)
7. For each discovered provider, call `createOrUpdateProvider(ctx, source, info)` -- collect errors
8. If authoritative mode: call `authoritativeSync(ctx, source, discovered)` to delete unmatched providers
9. Update status:
   - Set DiscoveredCount, ManagedCount, DiscoveredProviders list
   - Set LastSyncTime, LastSyncDuration, NextSyncTime
   - Set LastSyncError (join scan errors if any)
   - Set Synced condition: True if no errors, False with error details if partial/full failure
   - Set Ready condition based on overall health (True if sync succeeded, even partially)
10. Update metrics: DiscoverySourceCount, DiscoverySyncDuration
11. Status().Update
12. Return (ctrl.Result{RequeueAfter: refreshInterval}, nil)

**discoverProviders method -- route by type:**

**Namespace mode** (`discoverNamespace`):

- Build label selector from `source.Spec.NamespaceSelector.MatchLabels` and `MatchExpressions`
- List Namespaces matching labels via `client.MatchingLabels`
- Filter out ExcludeNamespaces (default: kube-system, kube-public, kube-node-lease)
- For each matching namespace, create a DiscoveredProviderInfo with:
  - Name: `<source-name>-<namespace-name>` (to ensure uniqueness)
  - Mode: ProviderModeRemote (namespace discovery implies service-level)
  - Source: `namespace/<namespace-name>`
- Return discovered map and any scan errors

**ConfigMap mode** (`discoverConfigMap`):

- Get the referenced ConfigMap (`source.Spec.ConfigMapRef.Name` in `ConfigMapRef.Namespace` or source namespace)
- Read the key (default "providers.yaml") from ConfigMap.Data
- Define a `ConfigMapProviderEntry` struct for YAML unmarshalling:

  ```go
  type ConfigMapProviderEntry struct {
      Mode     string `yaml:"mode"`
      Image    string `yaml:"image,omitempty"`
      Endpoint string `yaml:"endpoint,omitempty"`
      Command  []string `yaml:"command,omitempty"`
      Args     []string `yaml:"args,omitempty"`
  }
  ```

- Parse YAML as `map[string]ConfigMapProviderEntry` using `sigs.k8s.io/yaml`
- Convert each entry to DiscoveredProviderInfo
- Source: `configmap/<configmap-name>`

**Annotations mode** (`discoverAnnotations`):

- If PodSelector is set: list Pods matching labels, check for RequiredAnnotations (default: "mcp-hangar.io/provider"), extract provider name from annotation value
- If ServiceSelector is set: list Services matching labels, same annotation check
- For each annotated resource, create DiscoveredProviderInfo:
  - Name: annotation value or resource name
  - Mode: ProviderModeRemote
  - Endpoint: extract from annotations (e.g., `mcp-hangar.io/endpoint`) or construct from service/pod
  - Source: `annotation/<resource-kind>/<resource-name>`

**ServiceDiscovery mode** (`discoverServices`):

- List Services matching `source.Spec.ServiceDiscovery.Selector` labels
- For each service, find the port named `PortName` (default "mcp")
- Construct endpoint: `<protocol>://<service-name>.<namespace>.svc.cluster.local:<port>`
- Create DiscoveredProviderInfo with Mode=ProviderModeRemote
- Source: `service/<service-name>`

**createOrUpdateProvider method:**
Use `controllerutil.CreateOrUpdate(ctx, r.Client, provider, mutateFn)`:

- Set `LabelDiscoveryManagedBy` label to source.Name
- Apply template labels from `source.Spec.ProviderTemplate.Metadata.Labels`
- Apply template annotations from `source.Spec.ProviderTemplate.Metadata.Annotations`
- If template spec exists, deep copy it to provider.Spec
- Override with discovered values: Endpoint, Mode
- Set controller owner reference via `controllerutil.SetControllerReference()` if `source.ShouldSetController()`
- This handles BOTH create and spec drift overwrite (per locked decision: discovery owns the spec)

**authoritativeSync method (per locked decision -- scoped deletion):**

- List all MCPProviders in namespace with label `LabelDiscoveryManagedBy: source.Name`
- For each existing provider NOT in the discovered map, delete it
- Only operate on the discovered map from THIS sync (successfully-scanned sources only)
- If a discovery mode failed (its providers are NOT in the discovered map), those providers are NOT deleted
- Log each deletion
- Record events for deletions

**reconcileDelete method:**

- List all MCPProviders with `LabelDiscoveryManagedBy: source.Name`
- Delete each (owner references will handle GC, but explicit cleanup is cleaner)
- ClearDiscoveryMetrics
- Remove finalizer

**Filters application** (apply in discoverProviders after mode-specific discovery):

- If Filters.IncludePatterns set, only keep providers whose names match at least one pattern (use `regexp.MatchString`)
- If Filters.ExcludePatterns set, remove providers whose names match any pattern
- If Filters.MaxProviders set, truncate to that count

**SetupWithManager:**

```go
func (r *MCPDiscoverySourceReconciler) SetupWithManager(mgr ctrl.Manager) error {
    return ctrl.NewControllerManagedBy(mgr).
        For(&mcpv1alpha1.MCPDiscoverySource{}).
        Owns(&mcpv1alpha1.MCPProvider{}).
        Complete(r)
}
```

Note: Uses `Owns()` because discovery source creates MCPProviders with owner references. This auto-enqueues the source when owned MCPProviders change.
